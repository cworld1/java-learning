## Local Development

Environment requirements:

- [JDK](https://www.oracle.com/java/technologies/javase-downloads.html) 20.0.1+

### 1. Clone the repository

```shell
git clone https://github.com/cworld1/java-learning.git
cd java-learning
```

### 2. Run the code

```shell
javac HelloWorld.java
java HelloWorld
```
